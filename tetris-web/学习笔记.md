# Java → JavaScript 网页版俄罗斯方块：对照学习笔记

> 逐行对照 Java 版和网页版的核心差异，帮助你理解两种语言之间的转换思路

---

## 一、整体架构对比

| 方面 | Java Swing 版 | JavaScript 网页版 |
|------|-------------|----------------|
| 窗口/画布 | `JFrame` + `JTextArea[][]` | `canvas` + `requestAnimationFrame` / `setInterval` |
| 渲染方式 | 每个格子是一个 `JTextArea`，改背景色 = 更新格子 | 整块画布用 `canvas.fillRect()` 重绘 |
| 事件监听 | `KeyListener`（`keyTyped/keyPressed`） | `document.addEventListener('keydown')` |
| 定时循环 | `Thread.sleep()` + `while(true)` | `setInterval(fn, time)` |
| 弹窗 | `JDialog` | `HTML div` + CSS `display:flex/none` |
| 编译运行 | 先编译（javac），再运行（java） | 直接浏览器打开 HTML 文件 |
| 发布方式 | 打包成 JAR / EXE | 上传到 GitHub Pages，生成网址 |

## 二、变量声明的对照

### Java → JavaScript

```java
// Java
private static final int GAME_X = 26;        // 常量
int[][] data;                                  // 二维数组
JTextArea[][] text;
int score = 0;                                 // 整数
boolean isRunning = false;                     // 布尔
String name = "tetris";                        // 字符串
```

```javascript
// JavaScript
const GAME_X = 26;                             // 常量（用 const 更安全）
let data = [];                                 // 数组（动态大小）
let text = [];                                 // 未使用（改用 canvas）
let score = 0;                                 // 整数
let isRunning = false;                         // 布尔
const name = "tetris";                         // 常量字符串
```

**注意：** JavaScript 里没有 `int`、`boolean` 这种类型声明关键字，统一用 `let`（可修改）或 `const`（常量）。

## 三、循环结构的对照

### Java → JavaScript

```java
// Java：用两个嵌套 for 遍历 4x4 方块
int temp = 0x8000;
for (int i = 0; i < 4; i++) {
    for (int j = 0; j < 4; j++) {
        if ((temp & rect) != 0) {
            text[m][n].setBackground(Color.WHITE);
        }
        n++;
        temp >>= 1;
    }
    m++;
    n = n - 4;
}
```

```javascript
// JavaScript：几乎一样，只是没有类型声明
let temp = 0x8000;
for (let i = 0; i < 4; i++) {
    for (let j = 0; j < 4; j++) {
        if ((temp & rect) !== 0) {
            ctx.fillStyle = COLOR_EMPTY;
            ctx.fillRect((y + j) * BLOCK_SIZE, (x + i) * BLOCK_SIZE,
                          BLOCK_SIZE, BLOCK_SIZE);
        }
        temp >>= 1;
    }
}
```

**核心变化：**
- `text[m][n].setBackground(Color.WHITE)` → `ctx.fillRect(...)`
- 坐标从"格子索引"变成了"像素坐标"（乘以 BLOCK_SIZE）

## 四、游戏循环的对照

### Java：用线程休眠 + while 循环

```java
public void game_run() {
    ranRect();
    x = 0; y = 5;
    for (int i = 0; i < game_x; i++) {
        try {
            Thread.sleep(time);           // 线程等待（暂停时也在这里卡着）
            if (game_pause) {
                i--;                      // 暂停技巧：i 不增不减，等于原地踏步
            } else {
                if (!canFall(x, y)) {
                    // 落地处理...
                    break;
                } else {
                    x++;
                    fall(x, y);          // 下落一层
                }
            }
        } catch (InterruptedException e) { }
    }
}
```

### JavaScript：用 setInterval 定时器

```javascript
function gameLoop() {
    if (!isRunning || isPaused) return;  // 暂停时直接返回，不执行游戏逻辑

    if (!canFall()) {
        // 落地处理...
    } else {
        fall();                          // 下落一层
    }
}

// 每隔 time 毫秒执行一次 gameLoop
let gameLoopTimer = setInterval(gameLoop, time);
```

**暂停时的处理对比：**
- Java：让循环变量 `i--`，等下次进来还是同一行，看起来像暂停
- JavaScript：直接 `return`，定时器还在跑，但函数体不执行任何游戏逻辑

## 五、事件监听的对照

### Java：三种键盘方法

```java
public class Game extends JFrame implements KeyListener {
    @Override
    public void keyTyped(KeyEvent e) {
        // 处理字符键（P 键、空格）
        if (e.getKeyChar() == 'p') { togglePause(); }
        if (e.getKeyChar() == KeyEvent.VK_SPACE) { rotateBlock(); }
    }

    @Override
    public void keyPressed(KeyEvent e) {
        // 处理方向键
        if (e.getKeyCode() == 37) { moveLeft(); }   // ←
        if (e.getKeyCode() == 39) { moveRight(); }  // →
        if (e.getKeyCode() == 40) { moveDown(); }   // ↓
    }

    @Override
    public void keyReleased(KeyEvent e) { }          // 不用，留空
}
```

### JavaScript：统一的 keydown 事件

```javascript
document.addEventListener("keydown", function(e) {
    // P 键：暂停
    if (e.key === "p" || e.key === "P") {
        togglePause();
        return;
    }

    // 空格：旋转
    if (e.key === " ") {
        rotateBlock();
        return;
    }

    // 方向键
    switch (e.key) {
        case "ArrowLeft":  moveLeft();  break;
        case "ArrowRight": moveRight(); break;
        case "ArrowDown":  moveDown();  break;
    }
});
```

**关键区别：**
- Java 用 `KeyEvent.VK_SPACE` / `e.getKeyCode() == 37`（虚拟键码）
- JavaScript 用 `e.key === " "` / `e.key === "ArrowLeft"`（字符串，更直观）
- Java 的空格键在 `keyTyped` 里收到（因为会产生字符），JS 在 `keydown` 里收到

## 六、画布渲染的对照

### Java：用 JTextArea 组成网格

```java
// 创建 26x12 个 JTextArea 格子
text[i][j] = new JTextArea(game_x, game_y);
text[i][j].setBackground(Color.WHITE);  // 涂白
text[i][j].setBackground(new Color(0xD393FF));  // 涂紫色（边界）

// 变形时：清除旧位置，绘制新位置
clear(x, y);   // text[原位置].setBackground(WHITE)
drew(x, y);    // text[新位置].setBackground(COLOR_BLOCK)
```

### JavaScript：用 Canvas 按像素绘制

```javascript
// 初始化画布大小
canvas.width  = GAME_Y * BLOCK_SIZE;   // 12 * 20 = 240px
canvas.height = GAME_X * BLOCK_SIZE;  // 26 * 20 = 520px

// 每次重绘：清空整个画布，再画所有格子
function drawAll() {
    ctx.fillStyle = COLOR_EMPTY;
    ctx.fillRect(0, 0, CANVAS_W, CANVAS_H);  // 清空

    for (let i = 0; i < GAME_X; i++) {
        for (let j = 0; j < GAME_Y; j++) {
            if (data[i][j] === 1) {
                // 有方块，画填充矩形
                ctx.fillStyle = COLOR_BLOCK;
                ctx.fillRect(j * BLOCK_SIZE + 1, i * BLOCK_SIZE + 1,
                              BLOCK_SIZE - 2, BLOCK_SIZE - 2);
            } else if (j === 0 || j === GAME_Y - 1 || i === GAME_X - 1) {
                // 边界
                ctx.fillStyle = COLOR_BORDER;
                ctx.fillRect(j * BLOCK_SIZE, i * BLOCK_SIZE,
                              BLOCK_SIZE, BLOCK_SIZE);
            }
        }
    }
}
```

**两种思路的本质区别：**
- Java：把界面拆成一个个独立的小控件（JTextArea），每个控件独立管理自己的颜色
- JavaScript：只有一个大画布，用"清空 + 重绘"的方式，**每次都重画整个画面**

## 七、弹窗的对照

### Java：使用 JDialog

```java
// 创建模态弹窗
JDialog pauseDialog = new JDialog(this, "游戏暂停", Dialog.ModalityType.MODELESS);
pauseDialog.setSize(300, 380);
pauseDialog.setDefaultCloseOperation(JDialog.DO_NOTHING_ON_CLOSE);

// 显示 / 隐藏
pauseDialog.setVisible(true);   // 显示
pauseDialog.setVisible(false);  // 隐藏
```

### JavaScript：使用 HTML + CSS

```html
<!-- HTML：定义弹窗结构 -->
<div class="dialog-overlay" id="pauseDialog">
    <div class="dialog dialog-pause">
        <h2>⏸ 游戏已暂停</h2>
        <img src="pause.jpg" alt="暂停图片">
        <p>按 P 键继续游戏</p>
    </div>
</div>
```

```css
/* CSS：控制显示/隐藏 */
.dialog-overlay {
    display: none;             /* 默认隐藏 */
}
.dialog-overlay.active {
    display: flex;             /* 显示时变成 flex 布局 */
}
```

```javascript
// JavaScript：切换显示/隐藏
document.getElementById("pauseDialog").classList.add("active");    // 显示
document.getElementById("pauseDialog").classList.remove("active"); // 隐藏
```

**对比：**
- Java 的弹窗是一个独立窗口对象
- JavaScript 的弹窗是 HTML 元素，用 CSS 的 `display` 属性控制显隐
- Java 的弹窗有 `setFocusableWindowState(false)` 防止抢焦点；JS 不需要（因为页面只有一个焦点概念）

## 八、类与对象 → 原型与对象

### Java：完整的类

```java
public class Game extends JFrame implements KeyListener {
    private int score = 0;
    private boolean isRunning = false;

    public void togglePause() {
        this.isPaused = !this.isPaused;
    }
}
```

### JavaScript：直接使用变量和函数（不用 class 也可以）

```javascript
// 直接定义全局变量
let score = 0;
let isRunning = false;

// 直接定义函数
function togglePause() {
    isPaused = !isPaused;
}
```

或者用 ES6 class 写，更接近 Java 的风格：

```javascript
class Game {
    constructor() {
        this.score = 0;
        this.isRunning = false;
    }

    togglePause() {
        this.isPaused = !this.isPaused;
    }
}
```

## 九、部署到 GitHub Pages（让你的朋友通过链接玩）

### 第一步：创建 GitHub 仓库

1. 打开 https://github.com ，登录你的账号
2. 点击右上角 **"+"** → **New repository**
3. 填写仓库名称，比如 `tetris-game`
4. 选择 **Public**（公开仓库才能用 GitHub Pages）
5. 点击 **Create repository**

### 第二步：上传代码

1. 进入刚创建的仓库
2. 点击 **uploading an existing file**
3. 把 `tetris-web` 文件夹里的所有文件拖进去
4. 点击 **Commit changes**

> 注意：需要上传 `index.html`、`pause.jpg`、`gameover.jpg` 三个文件

### 第三步：开启 GitHub Pages

1. 在仓库页面，点击 **Settings**（设置）
2. 左侧菜单找到 **Pages**
3. **Source** 选项：选择 `Deploy from a branch`
4. **Branch** 选项：选择 `main`，文件夹选 `/ (root)`
5. 点击 **Save**

### 第四步：等待部署，找到你的游戏链接

几分钟后，刷新 Settings → Pages 页面，你会看到一行绿色的提示：

```
Your site is live at https://你的用户名.github.io/tetris-game/
```

把这个链接分享给朋友，他们就能直接点开玩了！

## 十、学习要点总结

| 概念 | Java 写法 | JavaScript 写法 |
|------|----------|----------------|
| 变量声明 | `int x = 0;` | `let x = 0;` |
| 数组 | `int[] arr = new int[10];` | `let arr = [];` |
| 循环 | `for (int i = 0; i < 10; i++)` | `for (let i = 0; i < 10; i++)` |
| 二维数组 | `int[][] data = new int[26][12];` | `let data = []; for(...) data[i] = [];` |
| 条件判断 | `if (a != 0)` | `if (a !== 0)` |
| 字符串比较 | `str.equals("abc")` | `str === "abc"` |
| 定时等待 | `Thread.sleep(500)` | `setInterval(fn, 500)` |
| 键盘事件 | `e.getKeyCode() == 37` | `e.key === "ArrowLeft"` |
| 类定义 | `public class Game { }` | `class Game { }` 或直接写函数 |
| 获取元素 | —— | `document.getElementById("id")` |
| 修改样式 | `component.setBackground(Color.RED)` | `element.style.background = "red"` |

---

> 💡 **最重要的转换思路：**
> Java 的 Swing 是"控件树"——每个 UI 元素都是独立对象；
> JavaScript 网页是"状态 + 渲染"——保持数据（state），随时可以重绘整个画面。
